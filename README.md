# Bot_vk
## About botvk

Этот бот для группы сайта Вконтакте.Этот бот работает для одной группы. Для простоты использования добавлены кнопки.Бот считывает какой сегодня день и чётность недели. С его помощью вы можете узнать:
 - Пары на сегодня
 - Пары на завтра
 - Время звонков
 - Пары на неделю
